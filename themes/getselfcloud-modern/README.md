# GetSelfCloud Modern (Paymenter Theme)

A modern Paymenter storefront theme inspired by KaasHosting and adapted with the visual direction from the `frontend` project.

## Features
- Gradient-rich modern landing experience
- Redesigned navigation, hero, product cards, and footer
- Mobile-friendly responsive layout
- Updated typography and button system
- Theme settings compatible with Paymenter defaults

## Development
- Build: `npm run build getselfcloud-modern`
- Dev: `npm run dev getselfcloud-modern`

## Installation
Copy these paths into your Paymenter installation root:
- `themes/getselfcloud-modern`
- `public/getselfcloud-modern` (compiled assets)

Then set theme in admin:
- `Settings -> General -> Theme -> getselfcloud-modern`
